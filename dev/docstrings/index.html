SPICEKernels.getkernel — Method
getkernel(identifier, type; to, cache)

Given the ephemeris file name, download the file and return a path to the file location. If a full URL is provided, that URL will be used. Otherwise, the identifier is assumed to be a SPICE General Kernel. If to is set, the file is downloaded to the provided path. If cache is enabled –- and it is by default –- the ephemeris file is downloaded to the SPICEKernels.jl scratch space.
